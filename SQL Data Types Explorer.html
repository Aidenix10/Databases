Pro Tip: Viewing Data Types in MySQL Workbench

Need to know exactly what data type a field is after running a query? You can easily check this right from the Results panel.

  • Look at the far right of the Result Grid panel at the bottom of the screen.
  • Click the Field Types tab (the magnifying glass icon) to see a list of every column in your result along with its exact data type, length, and flags.
  • Alternatively, you can always run DESCRIBE your_table_name; in your query editor.

Note: Exact storage limits and specific behaviors can vary slightly depending on the specific database system (MySQL, PostgreSQL, SQL Server, etc.).
