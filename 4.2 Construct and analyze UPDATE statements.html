The UPDATE Statement

An UPDATE statement is used to modify existing records in a table. It requires specific clauses to function correctly without destroying data.

Basic Syntax
UPDATE Students
SET FirstName = 'Kayden'
WHERE FirstName = 'Benjamin';

UPDATE [Table]

Specifies the target table. You cannot update multiple tables in a single standard UPDATE clause.

SET [Col] = [Val]

Determines which column changes and what the new value will be.

WHERE [Cond]

Crucial. Defines criteria. Without this, every row in the table updates.

Targeting Specific Data (WHERE Clause)

The WHERE clause acts as a filter. If the filter is specific (using AND), fewer rows are updated.

Scenario: Update Blue Shirts Only

UPDATE Products
SET Price = 49.99
WHERE Clothing = 'Shirt'
AND Color = 'Blue';

Result: Only 1 item updates because only one item is both a 'Shirt' AND 'Blue'.

Database Table State

Clothing Color Price Action
Shirt Red 44.99 Skipped
Shirt Blue 49.99 Updated
Pants Blue 44.99 Skipped

Mathematical Operations

You can use math within the SET clause to modify values relative to their current state.

Correct: 10% Increase

UPDATE Park
SET Ticket = Ticket * 1.1
  • Multiplies current price by 1.1 (110%).
  • Correct syntax order (`UPDATE Table`).
  • References the column (`Ticket`) to get the base value.

Common Errors

SET Ticket = .10

Error: Sets price to 10 cents (or 10% of value), not an increase.

SET Ticket = increase 10%

Error: SQL cannot parse natural language phrases like "increase".
